# מאגר פסקי דין - Psak Din Management System

מערכת מקיפה לניהול ואינדוקס חכם של פסקי דין של בית דין רבני, עם חיבור ישיר לספריית ספריא.

## ✨ תכונות עיקריות

### 📚 ניווט בספרות התורנית
- **גמרא** - גישה מלאה לכל הש"ס: 6 סדרים, כל המסכתות, דפים ועמודים
- **שולחן ערוך** - 4 חלקים: אורח חיים, יורה דעה, אבן העזר, חושן משפט
- **ספרים** - תנ"ך, משנה, רמב"ם, מפרשי רש"י, ספרי מוסר וקבלה

### ⚖️ ניהול פסקי דין
- העלאת קבצי HTML של פסקי דין
- חיפוש וסינון מתקדם
- קטגוריות ותיוג
- שמירה ב-localStorage לגישה מהירה

### 🧠 אינדקס חכם
- זיהוי אוטומטי של מקורות בפסקי דין (גמרא, שו"ע, רמב"ם)
- מבנה היררכי: סדר → מסכת → דף → עמוד
- חיבור דו-כיווני: מפסק למקור ומהמקור לפסקים
- תצוגה מפוצלת: גמרא מצד אחד ופסק דין מהצד השני

### 🌐 אינטגרציה עם ספריא
- טעינת טקסטים ישירות מ-API של ספריא
- שמירה אוטומטית לצפייה אופליין
- תמיכה במגוון ספרים ומקורות

## 🔍 שימוש

1. **העלאת פסקים**: לחץ על "העלאת פסקים" והעלה קבצי HTML
2. **חיפוש**: השתמש ב"אינדקס חכם" לחיפוש לפי מקורות
3. **גמרא**: נווט בגמרא ע"י בחירת סדר → מסכת → דף → עמוד
4. **שולחן ערוך**: בחר חלק וסימן
5. **ספרים**: בחר קטגוריה, ספר ופרק

## 📊 פורמט פסקי דין

הפסקים מתפרסים אוטומטית מקבצי HTML עם המבנה:
```html
<h1>פסק מספר XX - כותרת</h1>
<div>תוכן הפסק...</div>
```

המערכת מזהה אוטומטית:
- מספר פסק
- כותרת
- קטגוריות (ממונות, נישואין, גירושין וכו')
- מקורות (גמרא, שו"ע, רמב"ם)
